# News Portal Project

## Overview
A Django-based web application for publishing articles and newsletters. Readers can subscribe to publishers or journalists to receive personalised content feeds. Editors can approve content, and journalists can submit articles and newsletters.

## Features
- User roles: Reader, Journalist, Editor
- Publish articles and newsletters
- Review and approve content (Editor)
- Subscribe to publishers/journalists
- Personalised subscription feed for readers
- RESTful API endpoint for subscription feed

## Tech Used
- Python 3.12
- Django 5.2
- Django REST Framework
- SQLite (default, can switch to Postgres)
- HTML/CSS (with Django templates)
- X integration (optional posting)
