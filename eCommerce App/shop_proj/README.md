# Django Vendor Shop

A Django-based multi vendor eCommerce platform where:
- Vendors can create stores and products
- Buyers can browse products, add to cart, checkout, and leave reviews
- Orders generate email invoices
- New stores and products are optionally tweeted via the Twitter/X API

---

## Requirements

- Python 3.10+
- Django 4+
- Virtual Environment
